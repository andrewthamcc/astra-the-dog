# Astra the Dog

Paw-sonal website of Astra the Dog

## Description

Website that allows location to be found and shared via email or telephone. The options to share location only function given a set of query parameters present via QR code scan.

## Build

- Vite, React, TypeScript, Tailwind
- [LeafletJS & React Leaflet](https://leafletjs.com/)
- [SwiperJS](https://swiperjs.com/)
- [React Modal](https://github.com/reactjs/react-modal)
- Netlify
